# FocusDaily（今日焦点）
####介绍：今日焦点通过抓取凤凰新闻客户端的api接口，实现一个类似凤凰新闻客户端的新闻资讯软件，数据与凤凰新闻实时同步。包括图集、短视频、文字/视频直播等多种新媒体资讯。其中包含的频道一种40种，后续可能会添加更多。本项目仅用于交流学习，严禁用于商业。
